API Documentation

  • GET /api/drivers - Retrieves a list of all drivers.
  • GET /api/drivers/{id} - Retrieves a driver by their ID.
  • POST /api/drivers - Adds a new driver to the database. Requires driver details in the request body.
  • PUT /api/drivers/{id} - Updates an existing driver's details. Requires driver ID and new details in the request body.
  • DELETE /api/drivers/{id} - Deletes a driver from the database by their ID.

Teams Endpoints

  • GET /api/teams - Retrieves a list of all teams.
  • GET /api/teams/{id} - Retrieves a team by their ID.
